# ⚔️ KarenTalks – AI Debate Arena

KarenTalks is an interactive AI-powered debate platform where you argue, defend, and dominate against intelligent AI personalities.

Not your average chatbot — this is a **multi-agent argument simulator** with scoring, personalities, and real-time feedback.

---

## 🚀 What is KarenTalks?

KarenTalks lets you:

* Debate against different AI personalities
* Experience structured argument battles
* Get judged and scored in real time
* Improve your reasoning and persuasion skills

Think of it as:

> 🧠 ChatGPT + ⚔️ Debate Arena + 🎮 Game Mechanics

---

## 🧠 Features

### ⚔️ AI Personalities (Choose Your Opponent)

Each personality has a unique style:

* **🧘 Chill Sage** – Calm, wise, and composed
* **🔥 Riot Core** – Aggressive and relentless
* **😏 Witty Rebel** – Sarcastic and sharp
* **🧠 Cortex Prime** – Pure logic, zero emotion

---

### 🏆 Structured Debate System

* 5-round argument battles
* Turn-based responses
* AI judge decides winner each round
* Live scoreboard

---

### 📊 Performance Feedback

After each debate:

* Weak arguments identified
* Logical mistakes highlighted
* Actionable improvement tips

---

### 🎨 Dynamic UI System

* Dark / Light mode
* Personality-based themes
* Custom-designed chat bubbles
* Gradient-based modern UI

---

### 🧩 Smart State Management

* Tracks rounds, scores, and chat history
* Handles full debate lifecycle
* Seamless transitions between screens

---

## 🛠️ Tech Stack

* **Python**
* **Streamlit**
* **Google Gemini API**
* **Custom CSS (Advanced UI styling)**

---

## ⚙️ How It Works

1. Select a personality
2. Start the debate
3. Enter your argument
4. AI responds based on its personality
5. AI judge evaluates both sides
6. Scores are updated
7. After 5 rounds → final results + feedback
